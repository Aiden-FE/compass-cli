# @compass-aiden/cli

> Command line interfaces for Compass CLI

## Contributes

### Install

`pnpm install`

### Base commands

- `pnpm dev` 启用开发模式
- `pnpm build` 生产构建
- `pnpm lint` 代码校验
- `pnpm format` 代码格式化

### Publish library

1. 变更package.json内的version字段
2. 提交合并请求至master即可
